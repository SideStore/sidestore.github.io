SideStore

SideStore is an alternative app store that aims to make sideloading apps as seamless as possible. No jailbreak needed, and you don't need a computer after the initial install, only a WiFi connection.

Get started
Documentation

Apps

SideStore allows installation for any app you want, alongside the many available in our verified sources. And with the in-app JIT enabler, you can play hardware-accelerated games, emulators or whatever you want, wherever you want, with no computer required.

Open Source

SideStore is fully open-source and run by contributors. You can find the source code on GitHub, and contribute to the project if you want.
